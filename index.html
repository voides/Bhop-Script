How Does It Work?

Instead of making you download AutoHotkey and then downloading a script that will only work on flat surfaces, I've made my script in Visual Studio. Unfortunately, it will only work on Windows (sorry Mac users), and you will need Microsoft .NET Framework 4.5.2 or higher for it to work. The script works by using external cues on your PC while playing CS:GO to jump. Instead of injecting into CS:GO, or having the need to require numerous .dll files, it reads keystrokes, sound, and cursor movement while you hold down space. This is so the script can't get detected. No information from your computer will be sent anywhere, you wouldn't be able to use this offline if that was the case.


Is It Detected By VAC?

No. This can't be detected by VAC, even though it's a .exe program. The reason why, is because the script only modifies your keystrokes. It doesn't change the game at all. Which means that no offset updates, or CS:GO updates will stop the script from working. As long as you have "+jump" bound to space, you're good to go.

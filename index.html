How Does It Work?

Instead of making you download AutoHotkey and then downloading a script that will only work on flat surfaces, I've made my script in Visual Studio. Unfortunately, it will only work on Windows (sorry Mac users), and you will need Microsoft .NET Framework 4.5.2 or higher for it to work. The script works by using external cues on your PC while playing CS:GO to jump. Instead of injecting into CS:GO, or having the need to require numerous .dll files, it reads keystrokes, sound, and cursor movement while you hold down space. This is so the script can't get detected. No information from your computer will be sent anywhere, you wouldn't be able to use this offline if that was the case.

Which means that no offset updates, or CS:GO updates will stop the script from working. As long as you have "+jump" bound to space, you're good to go.

What Do The Different Modes Do?

I've included 3 different styles of bhopping into this script. Here's what they do:

  • Regular: All this does is make you jump as soon as you hit the ground. It only jumps once, so this style would be used on Valve hosted servers.
  • Scroll: This style is for CEVO, ESEA, Face It, or any other 3rd party platform. Simulates scrolling, so it appears as if you are scrolling to bhop if your commands and inputs are being checked by server admins.
  • Rage: This is probably the least expected mode to be on a bhop script. Basically this mode bhops perfectly for you every time. If there is no speed limit on the server, you will pass it since the script will land a perfect bhop every time. The other 2 styles make sure you don't go above 320 since that is the max speed on most CS:GO servers. This mode should not be used on Valve servers, since it will cause an Untrusted Account Ban. It's fine to use anywhere else though.

Bhop-script

CS:GO Bhop Script

How Does It Work?

Instead of making you download AutoHotkey and then downloading a script that will only work on flat surfaces, I've made my script in Visual Studio. Unfortunately, it will only work on Windows (sorry Mac users), and you will need Microsoft .NET Framework 4.5.2 or higher for it to work. The script works by using external cues on your PC while playing CS:GO to jump. Instead of injecting into CS:GO, or having the need to require numerous .dll files, it reads keystrokes, sound, and cursor movement while you hold down space. This is so the script can't get detected, no information from your computer will be sent anywhere. You wouldn't be able to use it offline if that was happening.


Is It Safe?

Yes, this program is 100% safe. Sadly, it will show up as a virus on your computer. But this is only because I didn't use an application protector on it. If you want to be sure it's safe to use, decompile the code and see for yourself. But if you don't want to download the script, you don't have to.
